# Gym Tracker - v0.1.0

## 🚀 Resumen
Primera iteración con foco en el submódulo de sesiones dentro de `log-workout`.  
Se trabajó principalmente en refactor, limpieza de código y mejoras en UX/UI para registrar entrenamientos.

---

## ✅ Cambios principales
- Refactor del módulo `session` en `log-workout`.
- Mejora de la interfaz de usuario para registrar sesiones.
- Se permite añadir notas a las sesiones (ejemplo: diferencias entre repeticiones planificadas y realizadas).
- Base para seed inicial con semanas 2 a 8, rutinas y ejercicios predefinidos.

---

## 📦 Estado actual del sistema
- **Log Workout**:  
  - Creación de sesión de entrenamiento.  
  - Registro de ejercicios, series, repeticiones y notas.  
  - Visualización básica de rutinas.  

- **Resto de módulos**:  
  En progreso, aún sin implementación completa.
